# QPon API

### [GET] /coupons

List all coupons

###### Params

|       name      |         description                                   |
|-----------------|-------------------------------------------------------|
|     `page`      |  Page number                                          |

---

### [GET] /coupons/search

Search coupon

###### Params

|       name      |         description                                   |
|-----------------|-------------------------------------------------------|
|     `keyword`   |  Keyword to match                                     |
|     `field`     |  Field to match. One of `title`.                      |
